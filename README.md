# Guruji Astro

## Technologies Used
- ReactJs
- SCSS
- React Icons
- React Slick

## Run Locally

Clone the project

```
  git clone https://github.com/aruncherkhiya/GurujiAstro-Frontend
```

Install dependencies

```
  npm i
```

Start the server

```
  npm run dev
```
